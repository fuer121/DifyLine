# Handoffs

这个目录用于沉淀 Dify 工作流改造、DSL 兼容性、导入排错和可复用方案。

## 当前文档

- [Dify DSL 工作流改造交接](./dify-dsl-workflow-redesign.md)：记录本次从角色形象分析工作流改造成角色定位分析工作流的过程、失败原因、兼容策略和后续迭代建议。

## 更新约定

后续同类任务优先更新已有 handoff，再新增文件。每次更新尽量记录：

- 目标 Dify 环境和版本线索
- 原始可导入 DSL 中已验证的节点类型
- 新增或失败的节点类型
- 浏览器 console / network 报错
- 最终可运行文件
- 下一步可迭代方向
